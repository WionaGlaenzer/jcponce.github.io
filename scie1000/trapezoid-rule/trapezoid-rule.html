Simulation

The following simulation shows a dynamic representation of the trapezoid rule to approximate the definite integral $$\int_a^bf(x) dx.$$

Things to try:

  • Change the function $f$. Example: sin(x^2-1)
  • Change the interval $[a,b]$.
  • Increase the number of trapezoids by dragging the slider n.
  • Active the box Value to see the approximation up to 5 decimal numbers.
