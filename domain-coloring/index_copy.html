Basic Tools: Real and Imaginary components

With the following tools you just need to enter the real and imaginary components of a complex function $f(z)=\text{Re}(x, y)+ i \text{Im}(x,y)$. Select a tool by clicking on the image.

HSB enhanced scheme

Plots level curves of the modulus and phase of $f(z)$.

HSB enhanced scheme

Plots level curves of the modulus, imaginary and real components of $f(z)$.

RGB enhanced scheme

Plots level curves of the modulus and phase of $f(z)$.

Advanced Tools: General complex functions

With the following tools you can plot an extensive variety of complex functions. You will need to use the sintax from Complex.js.

HSB enhanced scheme

Plots level curves of the modulus and phase of $f(z)$.

HSB enhanced scheme

Plots level curves of the modulus, imaginary and real components of $f(z)$.

RGB enhanced scheme

Plots level curves of the modulus and phase of $f(z)$.


Notes

The tools plot the enhanced phase portrait of a complex function $f(z)$ with modulus contour lines. The Hue-Saturation-Brightness (HSB) color scheme is used with the following equations:

\begin{eqnarray*} \text{H}&=&\frac{\pi - \text{atan2}(y, -x)}{2\pi},\\ \text{S}&=&1,\\ \text{B}&=&\frac15\left(\log_{1.3}\left(5 \sqrt{x^2+y^2}\right)-\text{floor}\left(\log_{1.3}\left(5 \sqrt{x^2+y^2}\right)\right)\right)+0.75 \end{eqnarray*}
where $x,y$ are the real and imaginary components, respectively. The p5 function colorMode(HSB, 1) is used in the tool.

Visit the Gallery containing different color schemes and alternative representations.

Things to try:

  • Change the real and imaginary components.

    Example 1: $f(z)=z^3-1$
    Re: pow(x, 3)-3*x*y*y-1
    Im: 3*x*x*y - pow(y, 3)

    Example 2: $f(z)=\log(z)$
    Re: log(sqrt(x*x + y*y))
    Im: atan2(y, x)

    Example 3: $f(z)=\sin(z)$
    Re: sin(x)*( ( exp(y) + exp(-y) ) / 2 )
    Im: cos(x)*( ( exp(y) - exp(-y) ) / 2 )

    List of available built-in functions is available here.
  • By default the function is plotted on the set given by $|\textbf{Re}\,z|\leq 4$, $|\textbf{Im}\,z|\leq 4 \cdot c = 2.9$, with $c=8/11$. To change this set enter a positive number in the input Box.
  • Click the button Update after every change to plot a new function.
