Basic Tools: Real and Imaginary components

With the following tools you just need to enter the real and imaginary components of a complex function $$f(z)=\text{Re}(x, y)+ i \,\text{Im}(x,y).$$

How to use the basic tool?
Notes
×

Level curves: Select an option 'Phase', 'Modulus', 'Phase/Modulus' or 'None'.

Re(x,y) and Im(x,y): Enter Real and imaginary components.

Examples:

For plotting $f(z)=z^2$, enter the values

Re(x,y)= x*x-y*y
Im(x, y)= 2*x*y

For plotting $f(z)=1/z$, enter the values

Re(x,y)= x+x/(x*x+y*y)
Im(x, y)= y-y/(x*x+y*y)

|Re z|<: Enter a positive real number defining the size of the square for plotting the function.

Save (png): Click to save your plot.

Notes about the basic tools

For these tools I used the enhanced phase portrait based on the work of E. Wegert [see 12]. The Hue-Saturation-Brightness (HSB) color scheme is used. For plotting the level curves of the modulus I used the following equations:

\begin{eqnarray*} \text{H}&=&\frac{\pi - \text{atan2}(y, -x)}{2\pi},\\ \text{S}&=&1,\\ \text{B}&=&\frac13\left(\log_{1.6}\left(\sqrt{x^2+y^2}\right)-\text{floor}\left(\log_{1.6}\left(\sqrt{x^2+y^2}\right)\right)\right)+0.7 \end{eqnarray*}
where $x,y$ are the real and imaginary components, respectively. The p5.js function colorMode() is used for all tools.

Others things to try:

  • Change the real and imaginary components.

    Example 1: $f(z)=z^3-1$
    Re: pow(x, 3)-3*x*y*y-1
    Im: 3*x*x*y - pow(y, 3)

    Example 2: $f(z)=\log(z)$
    Re: log(sqrt(x*x + y*y))
    Im: atan2(y, x)

    Example 3: $f(z)=\sin(z)$
    Re: sin(x)*( ( exp(y) + exp(-y) ) / 2 )
    Im: cos(x)*( ( exp(y) - exp(-y) ) / 2 )
