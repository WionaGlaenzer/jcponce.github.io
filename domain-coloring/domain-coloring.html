Domain coloring

The sketch below plots the enhanced phase portrait of a complex function $f(z)$ with modulus contour lines. The Hue-Saturation-Brightness (HSB) color scheme is used with the following equations:

\begin{eqnarray*} \text{H}&=&\text{atan2}(y, x),\\ \text{S}&=&100,\\ \text{B}&=&\log_{1.5}\left(5 \sqrt{x^2+y^2}\right)-\text{floor}\left(\log_{1.5}\left(5 \sqrt{x^2+y^2}\right)\right)+0.1 \end{eqnarray*}
where $x,y$ are the real and imaginary components, respectively. The values of H and B are mapped in p5.js onto the ranges $[0,360]$ and $[0,100]$, respectively.

Visit the Gallery containing different color schemes and alternative representations.

Things to try:

  • Change the real and imaginary components.

    Example 1: $f(z)=z^3-1$
    Re: pow(x, 3)-3*x*y*y-1
    Im: 3*x*x*y - pow(y, 3)

    Example 2: $f(z)=\log(z)$
    Re: log(sqrt(x*x + y*y))
    Im: atan2(y, x)

    Example 3: $f(z)=\sin(z)$
    Re: sin(x)*( ( exp(y) + exp(-y) ) / 2 )
    Im: cos(x)*( ( exp(y) - exp(-y) ) / 2 )

    Go to p5.js reference to learn more about the functions that are admitted.
  • The function is plotted by default in the set given by $|\textbf{Re}\,z|\leq 2.5$, $|\textbf{Im}\,z|\leq 2.5$. To change this set enter a positive number in the input Box.
  • Click the button Update after every change to plot a new function.
